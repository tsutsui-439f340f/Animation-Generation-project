# Animation generation project
本プロジェクトは2022/05/03に個人的に始めたプロジェクトです。
# 目的
本プロジェクトの目的は、テキストからアニメーションの作成を行うことです。
これは、様々なことに活用できると思っており、例えば、続編の作成がなくなってしまったアニメやオリジナルアニメの生成をしたり、MADの生成などが考えられます。
